# Android Emulators

<Callout type="info">
  Android emulator support is coming soon. Join the [early access list](https://console.limrun.com) to be notified.
</Callout>

Limrun Android emulators will give your coding agent the same cloud-native control surface as iOS — provision, build, install, and drive apps over a control WebSocket from any OS.

## Planned capabilities

- **Any API level** — Choose from Android 11 through 15, Pixel or generic AOSP profiles
- **ADB over WebSocket** — The agent issues `adb` commands without a local Android SDK
- **APK install** — Upload a prebuilt APK or let the Gradle sandbox compile and install
- **UI Automator bridge** — Read the accessibility tree and drive elements by resource-id or content-desc
- **Screen stream** — Live MJPEG stream accessible from any browser

## Stay updated

Sign up at [console.limrun.com](https://console.limrun.com) to get notified when Android support launches.
