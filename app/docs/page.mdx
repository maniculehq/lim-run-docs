# lim.run

Limrun gives cloud coding agents (Claude Code, Cursor, Codex) hands-on access to remote iOS simulators
with attached Xcode sandboxes. No local Mac. No `xcodebuild` running on the agent's host.

The agent issues `lim` commands, Limrun handles the build and the simulator, and you watch the device
stream live in a browser.

## What you can do

- **Remote iOS builds** — Run `xcodebuild` on a hosted Xcode instance from Linux, Windows, or CI
- **iOS simulator control** — Tap, type, scroll, take screenshots, and record video via the TypeScript SDK or CLI
- **MCP integration** — Each instance exposes a Model Context Protocol server, plug directly into Claude Desktop, Cursor, or Codex
- **Any agent, any OS** — The `lim` CLI and `@limrun/api` SDK work identically on every platform
